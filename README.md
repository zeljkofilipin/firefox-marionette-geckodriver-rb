# Firefox + Marionette + geckodriver

## Prerequisites

- [Firefox 47](https://www.mozilla.org/en-US/firefox/new/)
- [geckodriver](https://github.com/mozilla/geckodriver/)

## Installation

    gem install bundler
    bundle install

## Usage

    bundle exec ruby firefox.rb
